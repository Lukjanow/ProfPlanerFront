## Installation

1. Clone the repository
   ```sh
   git clone https://gitlab.rlp.net/top/24s/profplaner/pp_frontend_react.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build project & generate output.scss
   ```sh
   npm run build
   ```

4. Start project in development mode
   ```sh
   npm run dev
   ```
